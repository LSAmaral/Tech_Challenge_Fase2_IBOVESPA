# Tech Challenge - Fase 2 (Data Analytics)

## Objetivo
Modelo preditivo para prever tendência de alta/baixa do IBOVESPA com acurácia mínima de 75%.

## Resultados
- **Acurácia atingida:** 76.67%
- **Modelo:** Logistic Regression
- **Dados:** Histórico de 2020 a 2025

## Como rodar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o script principal: `python main.py`
